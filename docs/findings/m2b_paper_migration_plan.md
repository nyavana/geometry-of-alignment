## Context

§6 was deliberately written to the *minimum sufficient evidence* bar for the rank-1 weight intervention in §7: just (a) "L15 has the strongest refuse/comply contrast" and (b) "the contrast is approximately rank-1." Two figures support this — `signal_vs_layer.png` (Figure 6.1) and `pca_variance_per_layer.png` (Figure 6.2). Several other M2b deliverables exist on disk but are not cited or discussed. Some are mere variants of the chosen figures; some are real evidence that the section currently omits.

### B. UMAP/t-SNE layer grid — committed but never cited

Six figures, generated by `M2b 4.7` (commit `358abf5`):

- `results/figures/umap_layer_00.png`
- `results/figures/umap_layer_05.png`
- `results/figures/umap_layer_11.png`
- `results/figures/umap_layer_15.png`
- `results/figures/umap_layer_17.png`
- `results/figures/umap_layer_41.png`

These visualize refuse-vs-comply cluster separation across depth — exactly the qualitative evidence that L15 has a clean linearly-separable structure (and that L00 / L41 do not). The planning docs listed "UMAP activation visualization (M2b)" as a minimum deliverable, and the figures live in the repo, but no prose in `06_mechanistic.md` discusses them. **This is the largest real omission.**

**Suggested migration** (one new subsection in §6, ~150–250 words, between current Fig 6.1 and Fig 6.2):

> Add a single composite figure (2×3 grid of the six UMAPs, or pick 4: L00, L11, L15, L41) with a caption noting:
>
> 1. Early layers (L00, L05) show overlapping refuse/comply clouds — the contrast is not yet linearly representable.
> 2. Mid layers (L11–L17) show progressive cluster separation, with L15 the cleanest.
> 3. Late layers (L41) re-mix as the residual stream specializes for token prediction rather than category-level distinctions.
>
> Two-line interpretation: this is qualitative corroboration of the Cohen's *d* peak at L15 (Fig 6.1). It also pre-empts the obvious skeptical reading of Fig 6.2 — that the rank-1 dominance is an artifact of low signal — by showing the contrast is visibly bimodal at L15.

**Files to cite**: pick the four cleanest of the six. L00 / L11 / L15 / L41 is a reasonable narrative arc (pre-formation → mid-formation → peak → re-mixed).

**Effort**: ~30 min — pick figures, write caption, paste subsection. No new computation.

### C. Layer-by-layer numerical results — only inside PNG pixels

Section 6 cites point values (L15: Cohen's *d* = 2.868, top-1 PC fraction = 0.866; "L4–L17 band"), but the *per-layer* values for the full 42 layers exist only as rasterized curves inside the figure PNGs. No `results/mechanistic/*.csv` or `.json` table accompanies them.

**Suggested migration**:

1. Re-run or extract the 42-layer values from `results/activations/refusal_directions.pt` + `prompt_metadata.json` into a small JSON or CSV under `results/mechanistic/` (mirroring how `results/math_framework/mirsky_bound_per_layer.csv` was handled for §4).
2. Add a 3-column table to §6 listing the top ~5 peak layers by Cohen's *d* and their top-1 PC fraction.
3. Cite the CSV path in the figure captions for Figs 6.1 / 6.2 so a reader can recover the underlying numbers.

**Effort**: ~45 min — small extraction script + table prose. The extraction logic likely already exists inside `src/mechanistic/layer_analysis.py`; check before writing new code.
